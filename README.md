# Sequence Logos

![](output/sample_seq_logo_plot.png)

Takes input sequences from text file: 

```
TFFRLFNRSFTQALGK
PWEKIFLYAFVVALQL
PWEKIFLFAFLMALLF
PWEKIFLFAFLMALLF
...
PWEKIFLFAFLMALLF
PWEKIFLFAFLMALLF
WWEEIFMYAFLMALLF
WWKEIFLFAFLMALLF
```

Dependencies: 
- pandas
- matplotlib
- logomaker https://github.com/jbkinney/logomaker
